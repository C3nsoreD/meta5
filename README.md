# meta5.vim
This dark colorscheme, inspired by Tron

Designed for 256 color terminals
![meta5-screenshot](https://goo.gl/photos/JX7ukvP48do6SW8s7 "meta5-javascript")
![meta5-selection](https://goo.gl/photos/UrzykbqJcLev3bbP8 "meta5-select")

## Install
Pathogen
```
git clone https://github.com/christophermca/meta5.git
```

Vundle
```
Plugin 'christophermca/meta5'
```
